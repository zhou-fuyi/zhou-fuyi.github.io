开发环境准备

# 前言

本文用于说明本次开发所使用的环境，以及环境的搭建过程。

# 操作系统

Windows 10 专业版

其实我当时使用的操作系统的Arch Linux，开发完成后才又重装回Windows。

# 服务端

服务端使用Java语言进行开发，项目构建使用Gradle，开发工具使用Idea，服务发布使用Docker，下面是具体的版本：

  • JDK：OpenJDK 11
  • Gradle：7.3.2
  • Idea：IntelliJ IDEA 2022.1.3 (Community Edition)
  • Docker：Docker Desktop 4.9.1

# 小程序

  1. 首先需要自行完成小程序的注册，具体可以参考官方文档 (opens new window)。

  2. 其次，下载并安装小程序开发工具，参见下载页面 (opens new window)。

  3. 最后，调试基础库选择：2.24.6

# 参考

  • 开发者工具中无法显示echart或显示异常 (opens new window)

  • 在MobaXterm进入ssh终端后，键入上下左右出现乱码 (opens new window)

  • win10上修改docker的镜像文件存储位置 (opens new window)

  • WSL2 迁移 Docker 镜像存储位置 (opens new window)
